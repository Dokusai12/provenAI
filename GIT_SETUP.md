# Git Setup Instructions

After installing Git for Windows, run these commands in the provenAI directory:

## 1. Initialize Git Repository
```bash
git init
```

## 2. Add Remote Repository
```bash
git remote add origin https://github.com/Dokusai12/provenAI.git
```

## 3. Add All Files
```bash
git add .
```

## 4. Commit Changes
```bash
git commit -m "Setup Resend email configuration for contact form"
```

## 5. Push to GitHub
```bash
git push -u origin main
```

Note: You may need to authenticate with GitHub (username and personal access token).
